# WikiAgent 프로젝트 계획

## 목표
Andrej Karpathy의 LLM-Wiki 아키텍처를 구현하여 개인 지식 관리 자동화 시스템을 구축한다.

## 마일스톤

- Phase 1: 코어 엔진 개발 (파일 감시, 분류기, 위키 생성기)
- Phase 2: 대시보드 UI 개발 (D3.js 그래프 시각화)
- Phase 3: GitHub 동기화 자동화
- Phase 4: RL 정책 피드백 루프 완성

## 기술 스택

Node.js, Express, Chokidar, D3.js, simple-git

## 데드라인

2026년 6월 말까지 MVP 완성 목표
